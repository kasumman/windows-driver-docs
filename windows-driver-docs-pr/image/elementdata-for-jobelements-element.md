# ElementData for JobElements element


The required **ElementData** element contains the data that is returned for a job-related schema request.

## Usage

```xml
<wscn:ElementData Name="" Valid=""
  Name = "xs:string"
  Valid = "xs:string">
  child elements
</wscn:ElementData Name="" Valid="">
```
