# ADFBack element


The optional **ADFBack** element describes the capabilities of the back side of a duplex automatic document feeder (ADF) that is attached to the scanner.

## Usage

```xml
<wscn:ADFBack>
  child elements
</wscn:ADFBack>
```

## Attributes

There are no attributes.
