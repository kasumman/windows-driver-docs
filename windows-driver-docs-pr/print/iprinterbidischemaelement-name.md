# IPrinterBidiSchemaElement::Name method

The Name method returns the Bidi schema element name.

## Syntax

```cpp
HRESULT Name(
  [out, retval] BSTR *pbstrSchema
);
```

## Parameters

*pbstrSchema* \[out, retval\]  
The returned element name.

## Return value

This method returns an **HRESULT** value.
