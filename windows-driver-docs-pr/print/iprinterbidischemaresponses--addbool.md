# IPrinterBidiSchemaResponses::AddBool method

The AddBool method adds a new response of type BIDI\_BOOL to the collection.

## Syntax

```cpp
HRESULT  AddBool(
  [in] BSTR bstrSchema,
  [in] BOOL bValue
);
```

## Parameters

*bstrSchema* \[in\]  
The schema.

*bValue* \[in\]  
The new value.

## Return value

This method returns an **HRESULT** value.
