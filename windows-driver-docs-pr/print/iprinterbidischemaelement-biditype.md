# IPrinterBidiSchemaElement::BidiType method

The BidiType method returns the Bidi schema element type.

```cpp

  [out, retval] PrinterBidiSchemaElementType *pType
);
```

## Parameters

*pType* \[out, retval\]

This method returns an **HRESULT** value.
