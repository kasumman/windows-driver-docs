# IPrinterBidiSchemaResponses::AddFloat method

The AddFloat method adds a new response of type BIDI\_FLOAT to the collection.

## Syntax

```cpp
HRESULT AddFloat(
  [in] BSTR    bstrSchema,
  [in] FLOAT  fValue 
);
```

## Parameters

 *bstrSchema* \[in\]  
The schema.

 *fValue* \[in\]  
The new floating point value.

## Return value

This method returns an **HRESULT** value.
