# IPrinterBidiSchemaResponses::AddBlob method

The AddBlob method adds a new response of type BIDI\_BLOB to the collection.

## Syntax

```cpp
HRESULT AddBlob(
  [in] BSTR      bstrSchema,
  [in] IDispatch *pArray
);
```

## Parameters

*bstrSchema* \[in\]  
The schema.

*pArray* \[in\]  
The array.

## Return value

This method returns an **HRESULT** value.
