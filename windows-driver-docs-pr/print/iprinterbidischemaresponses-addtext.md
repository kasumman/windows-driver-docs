# IPrinterBidiSchemaResponses::AddText method

The AddText method adds a new response of type BIDI\_TEXT to the collection.

## Syntax

```cpp
HRESULT AddText(
  [in] BSTR bstrSchema,
  [in] BSTR bstrValue
);
```

## Parameters

*bstrSchema* \[in\]  
The schema.

*bstrValue* \[in\]  
The text.

## Return value

This method returns an **HRESULT** value.
