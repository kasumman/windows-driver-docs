# WDI\_TLV\_SSID


WDI\_TLV\_SSID is a TLV that contains an SSID.

## TLV Type


0x3B

## Length


The size (in bytes) of the array of UINT8 elements. An array length of 0 is allowed.

## Values


| Type      | Description                                        |
|-----------|----------------------------------------------------|
| UINT8\[\] | An array of UINT8 elements that specifies an SSID. |
