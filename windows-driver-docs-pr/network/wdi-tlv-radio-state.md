# WDI\_TLV\_RADIO\_STATE


WDI\_TLV\_RADIO\_STATE is a TLV that contains the state of the radio in hardware and software.

## TLV Type


0xA1

## Length


The sum (in bytes) of the sizes of all contained elements.
