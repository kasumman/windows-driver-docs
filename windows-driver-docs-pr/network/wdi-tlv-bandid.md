# WDI\_TLV\_BANDID


WDI\_TLV\_BANDID is a TLV that contains a band ID.

## TLV Type


0x39

## Length


The size (in bytes) of a UINT32.

## Values


| Type   | Description                  |
|--------|------------------------------|
| UINT32 | The identifier for the band. |
